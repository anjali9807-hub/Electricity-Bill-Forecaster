# Electricity Bill Forecaster

## Project Overview

This project predicts monthly electricity bills using Multiple Linear Regression.

## Features

- Units Consumed
- Average Daily Usage
- Number of Occupants
- AC Usage Hours
- Appliance Count

## Algorithm

Multiple Linear Regression

## Libraries

- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-Learn

## Results

The model predicts electricity bills based on historical usage patterns.
