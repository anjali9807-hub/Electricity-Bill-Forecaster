# Electricity Bill Forecaster Using Multiple Linear Regression

## Project Overview

The Electricity Bill Forecaster is a machine learning project that predicts monthly electricity bills using Multiple Linear Regression. The model analyzes historical electricity consumption data and estimates future electricity costs based on various factors influencing energy usage.

## Problem Statement

Predict monthly electricity bills using historical consumption data and household-related features. The project helps users understand electricity usage patterns and estimate future expenses.

## Features Used

* Units Consumed
* Average Daily Usage
* Number of Occupants
* AC Usage Hours
* Appliance Count

## Technology Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Machine Learning Algorithm

Multiple Linear Regression

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Correlation Heatmap Generation
5. Model Training
6. Prediction and Evaluation

## Evaluation Metrics

* Mean Squared Error (MSE)
* R² Score

## Results

The model successfully predicts electricity bills with high accuracy by identifying relationships between electricity consumption factors and billing amounts.
